# Table 

Supports custom tabke syntax for advanced table rendering

- rowspan, colspan
- cell width
- horizontal align
- header-rows
- header-columns
- multiple blocks, nested documents in cells

```table header-rows=2 header-cols=1
r2|
c2| Mars
c2| Venusp

| Produced
| Sold
| Produced
| Sold

| Teddy Bears
| $50,000
| $30,000
| $100,000
| $80,000

| Board Games
| $10,000
| $5,000
| $12,000
| $9,000
```
